Alguns comandos do Linux


ls - listar arquivos (semelhante ao dir no prompt do windows)

ls -la - lista com as permissões

ls -lh - lista com tamanhos

Configurar timezone

dpkg-reconfigure tzdata

Testar servidor web

Dentro do container web

curl http://localhost:80/index.php

Do host/desktop

curl http://localhost:8888/index.php

Download do adminer e baixar como index.php no diretório atual

curl -o index.php -OL https://github.com/vrana/adminer/releases/download/v4.7.8/adminer-4.7.8.php
