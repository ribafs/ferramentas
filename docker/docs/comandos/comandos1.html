Comandos 1

Listar images baixadas

docker images

Listar containers ativos

docker ps

Listar comandos inclusive inativos

docker ps -a

Listar somente os IDs dos containers

docker ps -qa

Efetuando login com o docker hub

docker login

docker login --username=yourhubusername --email=youremail@company.com

Copiar imagem

docker tag imagematual imagemnova

Para iniciar um container execute

docker start [container_name]

Para parar um container execute

docker stop [container_name]

Para reiniciar um container execute

docker restart [container_name]

Outros comandos


docker info 	Exibe informações sobre o sistems

docker inspect 	Retorna informações de baixo nível sobre objetos Docker

docker logout 	Log out do docker hub

docker logs 	Captura logs de um container

docker top      Mostra informações sobre os processos de um container rodando

docker update 	Atualizar configurações de um ou mais containers

docker update --cpu-shares 512 abebf7571666

docker update --cpu-shares 512 -m 300M abebf7571666 hopeful_morse

docker volume create hello

docker manifest  Gerencia manifestos de imagens docker

docker manifest COMMAND COMMAND

docker manifest create MANIFEST_LIST MANIFEST [MANIFEST...]

docker rename 	Renomear um container

Remover container

docker rm [container_name]

docker service ls
